# Netflix_project-mysql-
An end-to-end Data Analysis and Business Intelligence project using a comprehensive Netflix dataset. This repository showcases proficiency across the entire data lifecycle:

**Data Engineering (ETL)**: Implemented a robust ETL pipeline using Python to clean, transform, and structure the raw data.

**Database Management**: The cleaned and prepared data was efficiently stored and managed in a MySQL relational database.

**Data Quality & Analysis**: Conducted thorough data validation, including Outlier Detection and treatment, to ensure data integrity and reliable insights.

**Visualization & Reporting**: Developed a dynamic and interactive dashboard in Power BI to visualize key metrics like content trends, release patterns, and regional distribution for data-driven content strategy insights
